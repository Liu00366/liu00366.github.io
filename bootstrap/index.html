Bootstrap

Bootstrap is a front-end framework for rapidly developing Web applications and websites. Bootstrap an open source toolkit based on HTML, CSS and JavaScript.

Why use Bootstrap

Mobile first
Since Bootstrap 3, the framework has included a mobile first style throughout the library.
Browser support
All major browsers support Bootstrap.
Easy to get started
Once you have a basic knowledge of HTML and CSS, Bootstrap can be started
Responsive design
Bootstrap's responsive CSS ADAPTS itself to desktops, tablets, and phones. For more information on responsive design, see Bootstrap Responsive Design.

How to use Bootstrap

Step 1 - Use jsDelivr, a free open source CDN

Copy-paste the stylesheet into your before all other stylesheets to load our CSS.

CSS Use jsDelivr, a free open source CDN. Copy-paste the stylesheet <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous"> into your place before all other stylesheets to load the CSS.

JS Many of our components require the use of JavaScript to function. Specifically, they require our own JavaScript plugins and Popper. Place <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-MrcW6ZMFYlzcLA8Nl+NtUVF0sA7MsXsP1UyJoMp4YLEuNSfAP+JcXn/tWtIaxVXM" crossorigin="anonymous"></script>near the end of your pages, right before the closing tag, to enable them.
